# 🚀 Quick Start - Comece Agora!

## 1️⃣ Forma Mais Rápida (Sem Servidor)

```
1. Abra o arquivo index.html no seu navegador
2. Pronto! Funciona offline
```

✅ **Vantagem**: Simples e rápido
❌ **Desvantagem**: Pode ter limitações em alguns navegadores

---

## 2️⃣ Forma Recomendada (Com Servidor Local)

### No Linux/Mac:
```bash
cd "Site com abas interativas/vanilla"
bash server.sh
```

### No Windows (PowerShell):
```powershell
cd "Site com abas interativas\vanilla"
python -m http.server 8000
# Depois abra: http://localhost:8000
```

### Alternativas:
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# PHP
php -S localhost:8000

# Node.js
npx http-server
```

## 🆘 Problemas Comuns

**P: A página não carrega**
R: Certifique-se que está em um servidor HTTP. Tente `python -m http.server 8000`

**P: Estilos não aparecem**
R: Limpe o cache (Ctrl+Shift+Delete) ou use Ctrl+Shift+R

**P: JavaScript não funciona**
R: Abra o console (F12) e verifique se há erros
